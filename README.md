# PixelmonServer

친구 3~8명과 함께 즐기는 PvE 중심 Pixelmon RPG 서버.
VPS가 남는 기간 동안 **최소한의 개발로 최대한의 컨텐츠**를 제공하는 것이 목표.

---

## 서버 개요

| 항목 | 내용 |
|------|------|
| 규모 | 1인 운영 + 친구 3~8명 |
| 기반 | NeoForge + Pixelmon + SpongeNeo |
| 방향 | PvE 중심, 협력 플레이 |
| PvP | 선택적/이벤트성만 허용 |
| 핵심 컨텐츠 | 체육관, 퀘스트, 전설 레이드, 리그 |

---

## 디렉토리 구조

```
PixelmonServer/
├── docs/           # Phase 문서, 설계 메모
├── configs/        # 서버 설정 파일 (백업/버전 관리용)
├── scripts/        # 운영 스크립트 (시작/백업 등)
├── DESIGN.md       # 설계 문서 (기준 진실)
└── README.md       # 이 파일
```

---

## 설계 문서

- `DESIGN.md` — 서버 설계 및 로드맵 (기준 진실)
- `docs/` — Phase별 진행 기록
